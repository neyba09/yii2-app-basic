# Загрузка файлов

## Загрузка обложек книг

Реализована загрузка файлов для фото главной страницы книги.

### Хранение файлов

Файлы хранятся в файловом хранилище:
- Путь: `web/uploads/books/`
- URL: `/uploads/books/<filename>`

### Валидация

- Разрешенные форматы: JPG, JPEG, PNG, GIF, WEBP
- Максимальный размер: 5MB
- Валидация MIME-типов

### Использование

1. **Создание книги с обложкой:**
   - Заполните форму создания книги
   - Выберите файл изображения в поле "Фото главной страницы"
   - Сохраните

2. **Обновление обложки:**
   - При редактировании книги можно загрузить новое фото
   - Старое фото будет автоматически удалено
   - Если файл не выбран, старое фото сохранится

3. **Удаление книги:**
   - При удалении книги файл обложки также удаляется автоматически

### Технические детали

- Сервис: `app\services\FileService`
- Валидация: `app\requests\BookRequest`
- Обработка: `app\services\BookService`
- Имена файлов генерируются автоматически (уникальные)
